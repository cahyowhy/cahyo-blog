Compile File C# Dengan Menggunakan Cmd

20 November 2015

C# adalah bahasa program yang diciptakan oleh microsoft yang merupakan sebuah versi canggih dari c++. Umumnya bahasa c# lebih unggul dalam bidang OOP (oriented object programming) dari pada versi pendahulunya yaitu c++. bahasa c# sendiri mengandung 70% bahasa java, 20% bahasa c++, dan sisanya merupakan bahasa baru.

Bagi teman teman yang dulunya memang mahir menggunakan java, maka belajar c# pasti akan sangat mudah, karena hampir semua keyword c# mirip dengan java. selain itu sebagian fungsi rumusnya juga sebenarnya mirip dengan c++. saya sendiri ketika mau belajar c#, memang agak sulit dengan keywod yang aneh, aneh. Namun ketika dibaca lebih lanjut, sebenarnya memang agak mirip dengan c++ namun sifatnya lebih OOP lagi.

Nah bagi temen temen yang ingin belajar dasar dasar c# tanpa pusing pusing harus download aplikasi visual studio yang menuh menuhin laptop maka penulis rekomendasikan untuk menggunakan cmd sebagai kompiler dan notepad sebagai editornya. Nah, lalu bagaimana cara membuat agar cmd dapat mengcompile bahasa c#?...

1. Masuk ke explorer lalu masuk ke drive c:, masuk ke folder windows lalu microsoft. net, framework dan masuk ke folder 3.5 (C:\Windows\Microsoft.NET\Framework\v3.5).

2. Pastikan ada file yang bernama csc.exe di folder tsb

3. Masuk ke Computer--> Properties--> Advanced system settings--> Environment variables lalu cari Path

4. Salin lokasi file csc.exe di path, pastikan pada bagian belakan tanpa menggunakan spasi, contoh--> (bla bla bla;C:\Windows\Microsoft.NET\Framework\v3.5)

"System properties env variable path windows"

5. Tekan ok, lalu restart komputer.

6. Pastekan script hello world ini di notepad mu,

//keyword using
using System;
namespace hello_world {
//ini pendefinisian kelas
class halo {
// method
static void Main(string[] args){
// ini inti programnya
Console.WriteLine("Hello World");
Console.ReadKey();}
}
}
view raw check.cs hosted with ❤ by GitHub

7. Simpan dengan ekstensi .cs, (miss new2.cs) dalam drive misalnya di drive d: dalam folder cs.

8. Masuk CMD, lalu ketik d: untuk masuk ke drive d: lalu ketikan

cd cs
untuk masuk ke folder cs

9. lalu kompile file dengan csc namafile.cs

10. Buka file exe yang sudah di kompile di folder cs tadi
